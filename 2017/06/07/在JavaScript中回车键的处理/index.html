在JavaScript中回车键的处理

发表于 2017-06-07   |   分类于 Diary

在JavaScript中，处理字符串时要注意判断用户使用时是否会提交一些特殊符号，尤其是回车键。很多时候在提交时没有筛选，回车键随字符串一起存到了数据库里。在调用此数据时，很容易发生意外的错误。所以，在处理数据时，要多考虑用户的习惯，去除可能存在的隐患。

\r - Carriage Return(回车)

\n - Line Feed（换行）

这两种经常用的是\n ；因为\n能在 unix 和 Windows 中都正常显示.

在处理回车键时，一般有下列几种方式：

1、判断是否有/n字符，是否把/n替换成空字符

2、判断是否有\n回车符，是否把\n替换成空字符

3、判断是否有\r\n回车符，是否把\n和\r都替换成空字符
